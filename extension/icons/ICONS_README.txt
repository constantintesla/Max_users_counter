Иконки для расширения Chrome

Для работы расширения необходимы следующие иконки в формате PNG:
- icon16.png (16x16 пикселей)
- icon48.png (48x48 пикселей)  
- icon128.png (128x128 пикселей)

Вы можете создать простые иконки с помощью любого графического редактора или использовать онлайн-генераторы иконок.

Расширение будет работать и без иконок, но Chrome будет показывать стандартную иконку расширения.

Рекомендуется использовать простую иконку с текстом "Max" или символом чата.
